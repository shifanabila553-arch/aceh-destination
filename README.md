Deskripsi Proyek

Proyek Wisata Aceh Explorer adalah tampilan (views) statis yang dikembangkan sebagai Tugas Mandiri Ujian Tengah Semester (UTS) mata kuliah

.

Tujuan dari proyek ini adalah membuat tampilan landing page dan dashboard sederhana untuk mempromosikan destinasi wisata unggulan di Provinsi Aceh, dengan fokus pada pengembangan tampilan (frontend) statis menggunakan HTML dan Tailwind CSS.
